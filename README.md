## Build Setup

```bash
# clone project
$ git clone https://github.com/kasim393/ecommerce-react-firebase.git

# install dependencies
$ npm install

# change firebase config
$ utils/firbase/firebase.js

# serve with host at localhost:3000
$ npm run dev
```
